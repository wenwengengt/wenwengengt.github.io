MYSQL 语句

  • 个人博客
  • 前端拓展
  • 笔记本

包括几个方面：
注册，登录，增 删 改 查

后台php：

0. php 设置

设定要用的默认时区。自 PHP 5.1 可用：

1. 注册：—> 增 —-> insert into 表名 values （各项的值）

1.1 开启数据库；

1.2 提交； 通过$_POST提交

① 首先判断提交内容是否不为空（用户是否输入了内容）;
② 若用户都输入了内容，则提交到数据库：
  $sql = "书写数据库命令";
  $rs = mysql_query($sql);  -->  执行数据库命令
③ 判断是否提交到数据库：
  $rs 为 true 时，提交完成； 执行完成后的动作（跳转....）
  $rs 为 false 时， 提交失败； 执行失败后的动作（跳转....）

2. 登录： —->　查

2.1 开启数据库；

2.2 输入账号密码 , 登录(提交)：

① 首先判断提交内容是否不为空（用户是否输入了内容）;
② 若用户都输入了内容，则到数据库中去查询密码是否正确：
  首先要知道根据什么查询  ----->  根据用户名查询密码
  $sql = "书写数据库命令";
  $info = mysql_query($sql);  -->  执行数据库命令,得到查询到的内容
  $result = mysql_fetch_array($info, MYSQL_ASSOC);  ---->  设置得到的数据格式
③ 判断查询到的密码与用户输入的密码是否一致：
  如果一致则 为 true 时，
        登录成功； 执行完成后的动作（跳转....）
  如果不一致则 为 false 时，
        提示用户名或密码输入错误； 执行失败后的动作（跳转....）

3. 发布新闻：———> 增 （网数据库里添加）

4.修改新闻：——-> 先查询后更新 —–>　　先用 select 再用 update

4.1 开启数据库； —-> mysql_connect();

4.2 查 –> 把原来的数据在页面上展示出来：

首先要知道根据什么查询 —–> 根据传参（id…）查询
$sql = “书写数据库命令”;
$info = mysql_query($sql); –> 执行数据库命令,得到查询到的内容
$result = mysql_fetch_array($info, MYSQL_ASSOC); —-> 设置得到的数据格式
然后再把得到的数据在html上展示出来

4.3 更新 –> 把修改后的数据在数据库里更新.

$sql = “书写数据库命令”;
$rs = mysql_query($sql); –> 执行数据库命令
判断是否更新到数据库：
$rs 为 true 时，修改完新闻成； 执行完成后的动作（跳转….）
$rs 为 false 时， 修改失败； 执行失败后的动作（跳转….）

5. 删除新闻 ————> 删 （删除数据库的数据）——> delete from 表名 where 条件

5.1 开启数据库；

5.2 删除数据；

$sql = “书写数据库命令”;
$rs = mysql_query($sql); –> 执行数据库命令
判断数据库中的数据是否删除了：
$rs 为 true 时，删除成功； 执行完成后的动作（跳转….）
$rs 为 false 时， 删除失败； 执行失败后的动作（跳转….）

6. 上传图片： $_FILE();

$realName = $_FILES['pic']['name'];
$extPos = strrpos($realName, ".");  ///  从右面开始查找，找到 . 所在的位置，返回该位置的索引
$ext = substr($realName, $extPos);  ///  截取字符串，参数1是要截取的字符串，参数2是从什么位置开始截，返回截取的字符串
// 重命名  time() --> 获取当前的时间戳，  mt_rand(a,b); ---> 随机产生一个从a 到 b 之间的数；
 $picName = time().mt_rand(10000,99999).$ext;

 //move_uploaded_file(要移动的文件,文件的新位置); --> 把上传的文件移动到新位置，如果成功返回true，失败返回false
 $rs = move_uploaded_file($_FILES['pic']['tmp_name'], '../Public/Upload/'.$picName);
 // file_exists(文件的)  判断文件是否存在
 file_exists("../Public/Upload/" . $pic['pic_url'])

7. 平常纠错的几个步骤：

-1. 页面上报错的: 看在第几行，然后看他的上一行分号有没有漏掉，然后该行的变量有没有写错，
-2. 页面上没有报错的：
      +2.1 在$sql下面 打印输出一下$sql; 然后看一下命令有没有错，找不到可以在数据库软件里运行一下，如果能运行，说明没错，如果运行失败，说明命令行有错误；
      +2.2 在$rs = mysql_query($sql); 下面打印输出一下$rs;然后看一下他的值对不对;
      +2.3 在 $result = mysql_fetch_array($info, MYSQL_ASSOC); 打印输出一下$result;
      然后看一下他的值对不对；
